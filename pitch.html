Uso de Inteligencia Artificial

IA que entiende el contexto
lingüístico boliviano

🧠 LLaMA 3.3 70B via Groq
Modelo de lenguaje de última generación. Genera respuestas contextuales, detecta errores morfológicos y adapta el vocabulario al nivel del usuario.
🔥 Firebase (Google)
Auth anónima + Firestore para guardar el progreso de cada sesión. El usuario retoma donde lo dejó.
🎤 Web Speech API
Síntesis y reconocimiento de voz nativo del browser. Sin costo de API adicional.
✦ CORRECCIÓN GENERADA POR IA
Escribiste: Iskay mansana munani
Correcto: Iskay mansanata munani
En quechua el sufijo -ta marca el objeto directo. mansana + ta = mansanata.
